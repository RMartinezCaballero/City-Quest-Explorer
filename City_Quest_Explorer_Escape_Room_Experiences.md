# City_Quest_Explorer_Escape_Room_Experiences - Escape Rooms urbanos

Este archivo registra decisiones y estado del diseño para experiencias inmersivas en **City Quest Explorer**.

## Objetivo
Crear experiencias inmersivas para City Quest Explorer.

## Especialidades
- Gamificación
- Storytelling
- Turismo
- Psicología del juego

## Diseñar
- Historias
- Acertijos
- Pistas
- Recompensas
- Logros
- Misiones

## Plantilla de Misión (obligatoria)
Toda misión debe incluir:
- **Narrativa**
- **Objetivo**
- **Dificultad**
- **Duración**
- **Solución**

## Estado del proceso
- [ ] Definir framework de narrativa (arco, tono, referencias locales)
- [ ] Definir modelo de gamificación (puntos, pistas, tiempo, penalizaciones)
- [ ] Definir taxonomy de acertijos (lógica, observación, coordinación, QR, GPS)
- [ ] Definir formato de pistas (ocultas / parciales / hints) y reglas de desbloqueo
- [ ] Definir sistema de recompensas y logros (motivadores, rarezas, progresión)
- [ ] Definir estructura del “mission pack” por ruta/checkpoint

## Checklist de documentación (por cada elemento nuevo)
- [ ] Historia: resumen + objetivo narrativo
- [ ] Acertijo: tipo + mecánica + verificación
- [ ] Pista: nivel + condición de aparición + texto
- [ ] Recompensa: qué desbloquea + criterio + feedback UX
- [ ] Logro: nombre + descripción + umbral + persistencia (si aplica)
- [ ] Misión: completar plantilla obligatoria (Narrativa/Objetivo/Dificultad/Duración/Solución)
