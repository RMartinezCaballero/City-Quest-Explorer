# City_Quest_Explorer_Story_Writing - Historias inmersivas

Este archivo registra el diseño de historias inmersivas para City Quest Explorer, compatibles con **ciudades reales**.

## Objetivo
Crear historias tipo:
- National Treasure
- Da Vinci Code
- Sherlock Holmes
- Escape Rooms

## Diseñar
- Personajes
- Villanos
- Misterios
- Giros argumentales
- Audios
- Videos

## Regla de compatibilidad
- Las historias deben ser compatibles con ciudades reales.
- Usar referencias geográficas plausibles (lugares reales + rutas reales) y elementos culturales auténticos.

## Modelo de narrativa (plantilla)
Para cada historia/misión, definir:
- **Hook (1-2 frases):** gancho inicial.
- **Personajes:** protagonista(s) + roles + motivaciones.
- **Villano/Antagonista:** objetivo + método + conexión con la ciudad.
- **Misterio central:** qué se busca/resuelve.
- **Pistas:** dónde aparecen (checkpoint) y cómo se interpretan.
- **Giro(s) argumental(es):** qué se revela y en qué punto del progreso.
- **Clímax:** resolución en el último checkpoint.
- **Solución:** explicación narrativa y lógica del “por qué”.
- **Recompensas/logros narrativos:** qué se obtiene al final.

## Compatibilidad con ciudades reales
- [ ] Seleccionar ciudad objetivo (p.ej. Cartagena)
- [ ] Lista de ubicaciones reales (checkpoints)
- [ ] Elementos culturales (arte, historia, mitos urbanos)
- [ ] Validación de tono (no inventar “historia oficial”; usar ficción contextual)

## Audios y videos
### Audios
- [ ] Guion de locución (texto)
- [ ] Tiempos por pista (duración por segmento)
- [ ] Tono (misterio, aventura, humor)
- [ ] Alternativas por idioma (si aplica)

### Videos
- [ ] Tráiler de misión (30-60s)
- [ ] Clips de pistas (5-15s)
- [ ] Recap de final (15-30s)

## Checklist de documentación (por historia)
- [ ] Nombre de la historia + versión
- [ ] Ciudad real objetivo
- [ ] Personajes + ficha
- [ ] Villanos + ficha
- [ ] Misterios + enigma principal
- [ ] Pistas (por checkpoint): tipo, ubicación, texto/indicio
- [ ] Giros (cuándo ocurre + impacto en la solución)
- [ ] Audios (guiones/segmentación)
- [ ] Videos (storyboard básico)
- [ ] Solución final coherente
