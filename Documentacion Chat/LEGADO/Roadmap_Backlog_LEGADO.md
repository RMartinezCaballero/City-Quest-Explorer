# Fase 0 — Fundacional (4–6 semanas)

**Objetivo:** tener la base de datos, el panel admin y el contenido mínimo para no lanzar una app vacía.

| Épica | Historias de usuario clave |
|---|---|
| Infraestructura | Como equipo, necesito los 3 entornos (dev/staging/prod) en Supabase configurados con RLS base |
| Panel admin (CMS) | Como Administrador, quiero crear/editar/aprobar recetas con todos sus campos (historia, ciudad, ingredientes, pasos) |
| Roles | Como Administrador, quiero asignar roles (Editor, Chef, Historiador, Nutricionista, Fotógrafo, Moderador) a usuarios internos |
| Contenido fundacional | Como Editor, quiero cargar 30–50 recetas fundacionales con historia, ciudad y foto antes del lanzamiento |
| Marca | Como fundador, quiero la identidad visual (logo, colores, tipografía) lista para aplicar en todos los clientes |

---

# Fase 1 — MVP público (8–10 semanas)

**Objetivo:** app usable, con IA core y monetización básica activa.

| Épica | Historias de usuario clave |
|---|---|
| Auth | Como usuario, quiero registrarme con Google/Apple/Facebook o entrar como invitado |
| Catálogo y búsqueda | Como usuario, quiero buscar recetas por ciudad, dificultad, ingrediente o texto libre |
| Ficha de receta | Como usuario, quiero ver historia, origen, influencia árabe, ingredientes, pasos y valor nutricional de cada receta |
| Modo Cocina | Como usuario, quiero cocinar con pantalla siempre activa, temporizadores y avance de pasos por voz |
| IA — porciones y sustituciones | Como usuario, quiero ajustar porciones y sustituir ingredientes que no tengo |
| IA — nutrición | Como usuario, quiero ver calorías y alertas de alergias por receta |
| Lista de compras | Como usuario, quiero generar automáticamente la lista de compras de una receta |
| Mapa | Como usuario, quiero ver en un mapa de qué ciudad viene cada receta |
| Favoritos | Como usuario, quiero guardar mis recetas favoritas |
| Calificaciones | Como usuario, quiero calificar una receta después de cocinarla |
| Premium | Como usuario, quiero suscribirme a Premium y descargar recetas en PDF |
| Libro automático | Como usuario Premium, quiero generar mi Libro LEGADO en PDF con mi recetario personal |
| Pagos | Como usuario, quiero pagar con Wompi, Google Pay o Apple Pay |
| Modo offline | Como usuario, quiero acceder a mis recetas favoritas sin internet |

# Fase 2 — Comunidad e IA avanzada (3–4 meses)

| Épica | Historias de usuario clave |
|---|---|
| Contribuciones | Como usuario, quiero subir mi propia receta para revisión editorial |
| Comunidad | Como usuario, quiero comentar, dar like y seguir a otros cocineros/chefs |
| Gamificación | Como usuario, quiero ganar insignias y subir de nivel al completar retos |
| Retos | Como usuario, quiero participar en el reto "30 días cocinando costeño" |
| Chef IA conversacional | Como usuario, quiero preguntarle a Abuela Tula/Tío Roberto qué cocinar con lo que tengo |
| Reconocimiento de plato | Como usuario, quiero tomar una foto de un plato y que la IA me diga qué receta es |
| Menú semanal | Como usuario, quiero que la IA arme mi menú de la semana con mis favoritos |
| Cursos | Como usuario, quiero tomar cursos en video de cocina costeño-árabe |
| Podcast | Como usuario, quiero escuchar episodios cortos narrados por Tío Roberto |
| Historias y línea de tiempo | Como usuario, quiero explorar artículos sobre la inmigración árabe y su influencia culinaria |

# Fase 3 — Escuela, marketplace y datos (6+ meses)

| Épica | Historias de usuario clave |
|---|---|
| Certificados | Como usuario, quiero recibir un certificado digital al terminar un curso |
| Marketplace | Como usuario, quiero comprar especias, utensilios o libros recomendados desde la app |
| Experiencias físicas | Como usuario, quiero inscribirme a una visita guiada o evento presencial |
| Árbol genealógico de recetas | Como usuario, quiero ver cómo una receta cambió entre generaciones y ciudades |
| API pública B2B | Como partner (escuela/medio), quiero licenciar acceso al catálogo vía API |
| Realidad aumentada (exploratorio) | Como usuario, quiero ver el plato en 3D antes de cocinarlo *(validar demanda antes de construir)* |

---

# Backlog descartado / en pausa (documentado para no perder la decisión)

- **Publicidad:** descartada — el modelo es freemium sin ads.
- **Visión artificial de refrigerador:** descartada — reemplazada por input de texto/voz de ingredientes disponibles.
- **Citación automática de fuentes por IA:** descartada — la validación de fuentes históricas queda en el rol Historiador humano.
- **Árabe como idioma de lanzamiento:** en pausa — se evalúa en Fase 3 según tracción de la diáspora árabe-colombiana.
