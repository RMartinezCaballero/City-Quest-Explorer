# City_Quest_Explorer_UX_UI_Wireframes - UX/UI (móvil primero)

Este archivo registra el diseño UX/UI para City Quest Explorer, con enfoque en una experiencia tipo:
- Pokémon GO
- Geocaching
- Escape Rooms digitales

## Objetivo
Diseñar una experiencia similar (geolocalizada, gamificada y sencilla) con **móvil primero**.

## Prioridad
- Diversión
- Simplicidad

## Diseñar / Entregables
- Wireframes
- Flujos
- Componentes
- Sistema visual

## Principios de diseño
- Claridad inmediata (acciones obvias)
- Feedback en tiempo real (GPS/Progreso/Pistas)
- Menos fricción (1-2 pasos por acción)
- Accesibilidad (alto contraste, tipografías legibles, targets grandes)
